# Job Tracking App

A practical application designed to organize and track job applications. Built with **React 18** and **Redux Toolkit**, this project allows users to manage their job search process with status updates, filtering options, and a clean interface styled with **SASS**.


![jobtracker](https://github.com/user-attachments/assets/d3e30e6a-3df7-4326-b0c8-bd190fa058f5)

## Key Features

* **Global State Management:** Centralized data flow using **Redux Toolkit**.
* **CRUD Operations:** Create, Read, Update, and Delete job applications.
* **Status Filtering:** Filter jobs by status (Pending, Interview, Declined).
* **Mock Backend:** Realistic data persistence using **JSON Server**.
* **Custom Styling:** Modular CSS architecture using **SASS (SCSS)**.
* **Notifications:** User feedback provided via React Toastify.

## Tech Stack

| Category | Technologies |
| :--- | :--- |
| **Framework** | React 18 (Vite) |
| **State Management** | Redux Toolkit, React-Redux |
| **Routing** | React Router DOM v6 |
| **Styling** | SASS (SCSS) |
| **Backend** | JSON Server (Mock API) |
| **HTTP Client** | Axios |
